# 画像ファイル配置ガイド

このフォルダに施設の写真を配置してください。

## 必要な画像一覧

| ファイル名 | 用途 | 推奨サイズ | 説明 |
|-----------|------|-----------|------|
| `hero.jpg` | トップページ ヒーロー背景 | 1920×1080px 以上 | 施設の最も魅力的な写真（外観・リビングなど） |
| `concept.jpg` | コンセプトセクション | 800×600px 以上 | 和の雰囲気が伝わる室内写真 |
| `family.jpg` | ファミリーセクション | 800×600px 以上 | 子連れ・家族向けのほっこりした写真 |
| `gallery-1.jpg` | ギャラリー（大） | 800×600px | リビング・居間 |
| `gallery-2.jpg` | ギャラリー | 600×600px | 寝室 |
| `gallery-3.jpg` | ギャラリー | 600×600px | キッチン |
| `gallery-4.jpg` | ギャラリー | 600×600px | 外観・玄関 |
| `gallery-5.jpg` | ギャラリー | 600×600px | ダイニング・くつろぎスペース |
| `gallery-6.jpg` | ギャラリー | 600×600px | 浴室・バスルーム |
| `ogp.jpg` | SNSシェア用 OGP画像 | 1200×630px | サイトをシェアした時に表示される画像 |

## 画像の追加方法

### 1. 写真を配置する
このフォルダ（`public/images/`）にファイルをコピーしてください。

### 2. コードのコメントを外す
各ページファイルの中に以下のようなコメントがあります：

```
// 【写真追加時】上のdivを削除し、以下のコメントを外してください:
<Image
  src="/images/hero.jpg"
  alt="..."
  fill
  className="object-cover"
/>
```

- 上の `<div className="img-placeholder ...">` ブロックを**削除**
- `//` コメントを外して `<Image ...>` を有効化

## 画像の最適化について

- **形式**: JPG または WebP
- **品質**: 80〜90% で書き出すと軽量で高品質
- **スマホ対応**: 縦横比を考慮して、縦写真は避けるか別途調整

## 著作権について

Airbnb・Booking.com で使用している写真はそのまま使用可能ですが、
撮影者が別の場合は利用条件を確認してください。
